# sssb-studio

## Om projektet
Det här projektet tillhör Fanny Forsberg, digital kommunikatör på SSSB (Stockholms Studentbostäder).

## Vad vi bygger
Verktyg och webbsidor som hjälper Fanny i hennes dagliga arbete som digital kommunikatör.

## Tekniker
- HTML, CSS och JavaScript
- Ingen React, inget byggsteg — enkel och rak kod
- Kod sparas på GitHub
- Webbsidor publiceras med Vercel
- Data sparas i Supabase

## Kopplingar
- GitHub: aktivt
- Vercel: aktivt
- Supabase: aktivt
